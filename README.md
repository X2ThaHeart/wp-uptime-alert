# wp-uptime-alert
A C# app to easily monitor all your WordPress sites for uptime.

This app checks your server status, and returns it as active or error, and does the same for your WordPress site.

![Preview of WP uptime alert](https://github.com/X2ThaHeart/wp-uptime-alert/blob/master/preview.png?raw=true)

# Issues:
Sometimes the program will crash if you keep clicking with no input into the input box.

Will show error if wordpress site has no post feed available.

# Features
Clickable links from dataview

Server Uptime status

WordPress active post feed status

Time last checked (check every 5 minutes roughly)

Easy paste input of your domain list

Ability to stop program running & restart later

Loops continusly and can have more sites added anytime
